Tabla Comparativa

Característica Función principal Tipo de lenguaje Sintaxis básica Ubicación en el documento Relación
HTML (HyperText Markup Language) Estructura y contenido de la página web Lenguaje de marcado <h1>, <p>, <img>, <table>, <form> Dentro del archivo .html Es la base sobre la que CSS y Bootstrap aplican estilos y diseño.
CSS (Cascading Style Sheets) Estilo y presentación visual de la página web Lenguaje de estilos color: red;, font-size: 16px;, margin: 10px; Dentro del archivo .css o en la sección <style> del HTML Se usa para personalizar la apariencia de los elementos HTML.
Bootstrap Framework que facilita el diseño responsivo y estilizado Biblioteca basada en CSS y JavaScript class="btn btn-primary", class="container" Se incluye mediante <link> a la CDN o archivo local Bootstrap usa clases CSS que se aplican a elementos HTML para darles estilo y funcionalidad.
